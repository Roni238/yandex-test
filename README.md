# Тестовое задание для yandex-crowd

## Предпросмотр
https://roni238.github.io/yandex-test/

* Придерживался Pixel Perfect (макет кривой, так что значения везде странные, в паре мест костыльно использовал position: relative и сдвиги, например на макете в блоке about врапер картики повернут на 90 градусов, если ее развернуть как надо то от будет заходить на текст)
* Чтобы при ресайзе все выглядело нормально, тк на макете нет планшетов-мак экранов, на мобильную версию сделал max-width: 688px (+80px под центрование)
* В тестовом было указано "Кнопки на стартовом экране являются якорями", семантически это ссылки, так что сделаз их ссылками
* Для слайдера сделал универсальный скрипт, в конструктор передается родитель(в котором слайдер, кнопки итд) и опции (зациклить, автоплей итд)
* Следовал требованию - "используя стек html + css + чистый js (без библиотек и фреймворков)", решил будет без сборщика, без препроцессоров, шаблонизаторов и template, максимально ванильный стек
